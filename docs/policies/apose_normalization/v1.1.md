# A-Pose Normalization v1.1

## 정책 개요
A-Pose Normalization v1.1은 모든 신체 치수 측정의 전제 조건으로 사용되는
기준 포즈 정규화 정책이다.

본 정책은 입력 SMPL-X 바디를 측정 및 의류 피팅에 적합한
표준 A-Pose로 변환하며, 좌표계, dtype, 단위를 강제하여
downstream 측정의 재현성과 안정성을 확보한다.

## 정책 목적
- 측정 파이프라인 전반의 기준 포즈 단일화
- 팔/겨드랑이 간섭으로 인한 치수 측정 불안정 제거
- Shoulder, Girth, Height 등 모든 측정의 공통 전제 제공

## 적용 범위
- Phase 2 (Smart Mapper)
- Offline / Training
- 모든 신체 측정 정책의 upstream dependency
