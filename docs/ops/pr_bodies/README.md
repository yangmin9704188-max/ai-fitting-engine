# PR Body Drafts

이 폴더는 PR 본문 초안(draft)을 보관합니다.

## 목적

- PR 생성 전 본문 초안 작성 및 검토
- 운영 추적성: 과거 PR의 목적과 범위를 추적
- 템플릿 재사용: 유사한 PR의 본문을 참고

## 네이밍 규칙

- `pr_body_<feature_or_round>_<description>.md`
- 예:
  - `pr_body_round1_postprocess.md`
  - `pr_body_round2_kpi_diff.md`
  - `pr_body_ops_round_charter.md`
  - `pr_body_team_system_v0_2.md`

## 보관 정책

- **PR merge 후에도 보관**: 운영 추적성을 위해 삭제하지 않습니다.
- 루트에 임시 파일을 쌓지 않고, 이 폴더로만 관리합니다.
