# Obsidian Setup Guide

## 민영이 Obsidian에서 해야 할 UI 작업 (최소 6줄)

1. **파일 탐색기에서 Home 열기**: `docs/ops/OBSIDIAN_HOME.md` 더블클릭
2. **Graph view 켜기**: 왼쪽 사이드바에서 Graph view 아이콘 클릭 (문서 간 연결 관계 시각화)
3. **Canvas 열기**: `docs/ops/canvas/PROJECT_MAP.canvas` 더블클릭 (프로젝트 맵 뷰)
4. **파일 검색 사용**: `Ctrl+O` (또는 `Cmd+O` on Mac)로 빠른 파일 검색
5. **(선택) 스타(즐겨찾기)로 Home 고정**: `docs/ops/OBSIDIAN_HOME.md` 우클릭 → "Add to favorites" (빠른 접근)
6. **링크 따라가기**: Home 또는 Canvas에서 원하는 문서 링크 클릭하여 탐색

## LLM 세션에 첨부할 최소 3파일

LLM과의 협업 세션에서 프로젝트 컨텍스트를 제공하기 위해 다음 3개 파일을 첨부하세요:

1. **OBSIDIAN_HOME.md**: 프로젝트 탐색의 단일 진입점, 5-Layer 네비게이션, Baseline, Registries 링크
2. **measurements/INDEX.md**: 측정 정책 SoT 인덱스, Current SoT, Legacy 문서 목록
3. **sizekorea_integration_checklist.md**: SizeKorea 통합 상태 체크리스트, Integration Map, Evidence Anchors

이 3개 파일로 프로젝트의 핵심 구조와 현재 상태를 빠르게 파악할 수 있습니다.
